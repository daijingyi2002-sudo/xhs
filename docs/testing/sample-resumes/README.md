# 测试简历样本

以下 5 份简历均为完全虚构的测试数据，仅用于产品功能验证，不对应任何真实个人。

设计目标：

- 年龄分布覆盖 17-30 岁
- 背景尽量拉开差异，便于测试推荐、画像、追问、面试与简历优化链路
- 内容写得足够像真实简历，但不包含真实隐私

文件列表：

- `01-林见夏-17岁-高中生-AI内容运营探索.md`
- `02-周子勋-19岁-高职生-电商运营转AI产品.md`
- `03-沈一宁-22岁-心理学本科-用户研究型产品.md`
- `04-陈拓-26岁-土木工程转互联网产品.md`
- `05-顾晴川-29岁-临床护理转医疗AI产品.md`

建议测试方式：

- 用 `01` 测“低龄、经历少、潜力型用户”
- 用 `02` 测“高职/专科、电商实操、数据意识”
- 用 `03` 测“研究能力强、产品方法较强”
- 用 `04` 测“跨行业转岗、项目落地与复杂协同”
- 用 `05` 测“成熟转岗、行业理解强、AI理解待补”
