# XHS Job Agent

An editorial, agent-first web product for AI product manager job discovery, fit analysis, mock interviews, and resume optimization.

## Stack

- Node.js 22
- npm workspaces
- Next.js App Router
- Supabase
- PostgreSQL + pgvector
- LangGraph service scaffold
- Playwright crawler scaffold

## Workspaces

- `apps/web`
- `packages/domain`
- `packages/ui`
- `packages/config`
- `services/ai`
- `workers/crawler`

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
